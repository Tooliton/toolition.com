## Stack

- [Next.js](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

The site will be available at [http://localhost:3000](http://localhost:3000).

## Features

- Hero section with animated code snippet and clear value proposition
- Featured libraries grid with quick access to GitHub repositories
- Philosophy, open source, docs preview, and CTA sections
- Responsive design with a dark, minimal aesthetic inspired by modern developer tooling brands
- SEO-friendly metadata and share image

## Deployment

Build the production bundle with:

```bash
npm run build
```

Then start the production server:

```bash
npm run start
```
